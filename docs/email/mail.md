## SECTION 0 — 오늘의 핵심 (30초 요약)

**데이터 소스:** `sonar_context.key_narrative` + 이상 움직임 자동 감지 키워드

**역할:** 바쁜 독자가 이것만 읽어도 오늘 시장의 핵심을 파악할 수 있는 3문장 이내 요약.

```
오늘 시장의 핵심 내러티브 한 줄
(sonar_context.key_narrative 직접 사용)

이상 움직임 감지 시 추가 노출:
· [VIX > 25] 변동성 경보 — VIX 28.3 (임계값 25 초과)
· [US10Y 변동 > ±1%] 10년물 금리 4.82% (+0.12%p) 급등
· [개별주 ±3% 초과] NVDA -4.2% / META +3.1%
· [BTC ±3% 초과] BTC +3.8% 급등
```

**분기 규칙:**
- 이상 움직임 없는 날: 내러티브 1줄만 노출, 이하 블록 생략
- `data_quality.status=degraded`: "일부 데이터 지연/누락 있음" 배너 상단 삽입
- `data_quality.status=critical`: "오늘 브리핑은 데이터 품질 이슈로 일부 수치가 누락됩니다" 고정 경고 삽입

## SECTION 4-1 — 이슈 브리핑 (토픽별 서술)

**데이터 소스:** `topic_summaries[*].summary_text` + `topic_summaries[*].market_implication` + `topic_summaries[*].key_data_points`

**형식:** Bloomberg 스타일 서술 단락, 토픽당 3~5문장, 수치 인라인 삽입.

```
[ 거시 / 금리 ]
연준 위원들의 발언이 엇갈리는 가운데, 10년물 국채금리는 4.52%로
전일 대비 8bp 상승했다. 시장은 이를 5월 동결 가능성의 재확인으로
해석하고 있으며, 달러 인덱스(DXY 104.2)는 소폭 강세를 유지했다.
오늘 21:30 발표되는 소매판매가 예상치(+0.3%)를 상회할 경우
추가 금리 인상 기대가 재점화될 수 있어 주목된다.

[ AI / 빅테크 ]
(동일 구조)

[ 비트코인 ]
(동일 구조)

[ 미국 증시 ]
(동일 구조)
```

**생성 규칙:**
- `topic_summaries` 존재 시: `summary_text` + `market_implication` 조합으로 서술
- `topic_summaries` 없는 토픽: 해당 토픽의 `news` 항목으로 직접 서술 생성
- `key_data_points`의 수치는 서술 문장 안에 자연스럽게 녹여 넣음 (표 형태 사용 안 함)

## SECTION 4-2 — 오늘의 핵심 뉴스 5선

**데이터 소스:** `packet["news"]` — `source_tier` 오름차순, `age_hours` < 24 우선

**형식:** 번호 + 헤드라인 + 서술 단락 + 원문 링크. 건당 3~4문장.

```
① 엔비디아, 중국 수출 추가 제한 검토 — WSJ
   바이든 행정부가 AI 칩 수출 허가 요건을 강화하는 방안을 검토
   중이라고 WSJ이 보도했다. 이번 조치가 현실화될 경우 엔비디아의
   중국향 매출(전체의 약 15%)이 직접 영향권에 든다.
   AMD, AVGO 등 동종 업체 역시 동반 하락 압력을 받을 수 있다.
   → 원문 링크

② ~ ⑤ 동일 구조
```

**표시 규칙:**
- `official_source=True`: 헤드라인 앞 `[공식]` 태그 삽입
- `source_tier=1` (Reuters/Bloomberg/WSJ/FT/CNBC/CoinDesk): 출처 볼드 처리
- `why_it_matters` 있는 기사: 해당 내용을 단락 마지막 문장에 흡수
- `source_tier` 높은(신뢰도 낮은) 기사: "~로 알려졌다", "~로 전해졌다" 등 표현 조정
- `age_hours > 24`: 하단으로 밀거나 5건 초과 시 제외
- 최대 5건 고정

## SECTION 4-3 — 섹터 / 자산 영향 매핑

**데이터 소스:** `topic_summaries[*].notable_stocks` + `news[*].why_it_matters` + `x_market_signals`

**역할:** 오늘 뉴스가 어떤 자산·섹터를 실제로 움직이는지 수혜/압력/관망 3분류로 정리.

```
오늘 주목 흐름

수혜 방향 (+)
  TLT (장기채 ETF)    금리 인하 기대 재부상 시 수혜
  META, AMZN          광고·커머스 매출 전망 상향 보고서 영향
  IBIT, FBTC          기관 BTC 매수세 지속, ETF 순유입 확인

압력 방향 (-)
  NVDA, AMD, SOXX     중국 수출 규제 확대 우려
  DXY (달러)          연준 동결 기대 강화 시 약세 전환 가능성

중립 / 관망
  SPY, QQQ            소매판매 발표 전 방향성 탐색 중
```

서술 보강 (매핑 아래 1~2문장):
```
반도체 섹터는 수출 규제 뉴스와 SOXX +0.3% 간의 괴리가 주목된다.
선물 시장은 아직 규제 현실화 가능성을 낮게 반영하고 있는 것으로
해석되며, 오늘 소매판매 결과가 오후 방향성을 결정할 가능성이 높다.
```

**생성 규칙:**
- `notable_stocks`에 없는 자산은 `why_it_matters` 키워드 파싱으로 보완
- 수혜/압력 판단 근거 한 줄 서술 필수 (근거 없는 매핑 금지)
- 3분류 모두 최소 1개 이상 항목 있어야 섹션 생성 (없으면 생략)

## SECTION 5-1 — 주간 맥락 연결

**데이터 소스:** `sonar_context.analyses` + `topic_summaries[*].market_implication` + 요일 기반 고정 틀

**역할:** 오늘 하루 데이터를 이번 주 흐름 속에서 위치시키는 서술. 이벤트 캘린더(Section 6)와 보완 관계 — 중복 없이 연결.

```
이번 주 흐름 속 오늘 (3/18 화)

이번 주는 FOMC 의사록(수), PCE(금)을 앞두고
데이터 확인 전 포지션 조정 국면으로 시작됐다.
월요일 증시 하락(-0.8%)은 지난주 고용지표 서프라이즈의
여운이 지속된 결과로 해석된다.

오늘(화) 소매판매 발표가 이번 주 첫 번째 방향 트리거다.
  예상치(+0.3%) 상회 시 → 금리 인하 기대 후퇴 → 성장주 추가 압력
  예상치 하회 시 → 경기 둔화 우려 + 인하 기대 재부상 → 채권 강세

수요일 FOMC 의사록에서 매파 온도가 확인되는지 여부가
이번 주 증시 방향성의 최종 결정 변수다.
```

**요일별 고정 서술 틀:**

| 요일 | 포지셔닝 프레임 |
|------|----------------|
| 월 | 지난주 마감 여운 + 이번 주 이벤트 예고 |
| 화~목 | 이벤트 전후 위치 + 오늘 트리거 |
| 금 | 이번 주 결산 + 다음 주 변수 예고 |

**생성 규칙:**
- `sonar_context` 있을 때: analyses + market_implication 조합
- `sonar_context` 없을 때: `topic_summaries[*].market_implication`만으로 구성
- 시나리오 분기(상회/하회 시)는 오늘 발표 예정 지표가 있을 때만 삽입

## SECTION 5-2 — Sonar 교차 분석

**데이터 소스:** `sonar_context.analyses[*]` — 최대 3건

**형식:** 주제 레이블 + 서술 단락 4~6문장. 마지막 문장은 반드시 "다음에 볼 것" 또는 "주의 조건" 제시.

```
[ 분석 1: 금리와 BTC의 비전형적 동조 ]
통상 금리 상승 국면에서는 위험자산인 BTC가 하락 압력을 받는다.
그러나 오늘 10년물 금리가 상승했음에도 BTC는 2.1% 올랐다.
이는 기관 ETF 수요가 매크로 변수를 일시적으로 압도하고 있음을
시사한다. BlackRock IBIT의 일일 순유입(+1,240 BTC)이 이를 뒷받침한다.
해당 흐름이 지속되려면 ETF 유입이 꺾이지 않아야 하며,
공포탐욕지수가 75를 넘을 경우 단기 차익실현 압력이 높아질 수 있다.

[ 분석 2: ... ]
[ 분석 3: ... ]
```

**생성 규칙:**
- `cross_sector_link`는 별도 항목 분리 없이 서술 중간에 자연스럽게 녹여 넣음
- `sonar_context` 없으면 섹션 전체 생략
- 분석 건수: `analyses` 배열 길이 그대로 사용 (최대 3건 제한)

## SECTION 5-3 — X 시장 반응

**데이터 소스:** `x_market_signals[*]`

**형식:** 개별 시그널 나열 대신, 전체 센티먼트 흐름을 2~3단락으로 압축 서술. 계정 핸들 인라인 삽입.

```
[ X(트위터) 실시간 반응 ]

오늘 X에서 형성된 시장 담론은 대체로 매크로 불확실성에
집중됐다. 복수의 시장 참여자들이 소매판매 결과를 앞두고
관망을 권고했으며(@계정A, @계정B), 반도체 섹터에 대해서는
단기 반등 매수 기회라는 시각(@계정C)과 추가 하락 가능성
경고(@계정D)가 팽팽하게 맞섰다.

BTC 관련해서는 ETF 유입 지속을 근거로 한 강세 시각이 우세했으며,
```
