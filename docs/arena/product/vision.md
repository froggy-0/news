# BTC Signal Arena — 비전 및 장기 목표

> 최초 작성: 2026-06-19
> 성격: 비기술 기획서 (전략·방향·철학)

---

## 1. 한 줄 정의

**"결과를 숨기지 않는 BTC 알고리즘 신호 서비스"**

수익 난 거래만 골라 보여주는 기존 서비스와 달리,
모든 손실·실패 기록을 공개하고 시간 순으로 블록체인처럼 쌓아가는
투명한 알고리즘 트랙레코드 플랫폼.

---

## 2. 왜 만드는가 (문제 정의)

### 2-1. 시장의 신뢰 문제

크립토 시그널 시장의 현실:
- 텔레그램 채널 수천 개가 "정확도 90%"를 자랑하지만 아무도 검증 불가
- 손실 콜은 조용히 삭제, 수익 콜만 캡처해서 홍보
- 사용자는 돈을 잃고 나서야 속았다는 걸 깨달음

결과: **크립토 시그널 시장 전체에 대한 근본적 불신**이 팽배하다.

### 2-2. 기존 퀀트 서비스의 진입 장벽

- 글로벌 퀀트 서비스(Quantopian, QuantConnect 등): 백테스트 기반, 미래 성과 보장 없음
- 한국 크립토 커뮤니티: 정성적 "전문가 견해" 위주, 정량적 실적 공개 서비스 없음
- 개인 투자자: 알고리즘을 직접 검증할 코딩 능력도, 시간도 없음

---

## 3. 솔루션

### 핵심 원칙 하나: **기록은 지울 수 없다**

```
신호 발생 → Supabase 기록 (UTC 타임스탬프) → 신호 반전·flat 전환·스톱로스 시 청산
             ↑
             이후 수정 불가. 손실이어도 공개.
```

모든 거래 기록이 생성 시점부터 불변(immutable)하게 저장되고,
누구나 조회할 수 있는 리더보드로 실시간 공개된다.

### 무엇을 제공하는가

| 제공 가치 | 설명 |
|---|---|
| **투명한 트랙레코드** | 모든 진입·청산 기록이 날짜·가격과 함께 공개 |
| **멀티 알고리즘 비교** | 5개 전략이 동일 시장에서 동시에 경쟁, 편향 없이 비교 |
| **거시 + 기술 복합 분석** | VIX, FNG, Regime State + RSI, MACD, BB 지표 통합 |
| **실시간 손절 반영** | ATR 기반 `stop_loss_price`를 실제 Binance 시세로 실시간 체크 |
| **구독 불필요 (초기)** | 데이터를 쌓는 단계에서는 무료 공개 |

## 4. 목표 사용자

### Primary: 데이터 지향 개인 크립토 투자자

- 자체 분석 능력은 있지만 거시·기술 지표를 통합할 시간이 없는 사람
- "왜 이 신호인가"를 이해하고 싶은 사람 (블랙박스 AI 신호 거부감)
- 한국어 커뮤니티 (디시인사이드 코인갤, 클리앙, 카카오 오픈채팅 등)

### Secondary: 퀀트 지망 학습자

- 알고리즘 포워드 테스트 결과를 직접 분석하고 싶은 사람
- "이 알고리즘이 왜 이 시장에서 실패했는가"를 공부하고 싶은 사람

### 미래: 기관·팀 단위 사용자

- 자체 알고리즘을 아레나에 올려 공개 검증받고 싶은 운용사·팀

## 5. 핵심 차별화 요소

| 경쟁 요소 | 일반 크립토 시그널 | **BTC Signal Arena** |
|---|---|---|
| 과거 기록 투명성 | 없음 (선택적 공개) | 완전 공개, 삭제 불가 |
| 알고리즘 논리 공개 | 없음 | 조건 명시 (예: VIX < q40 AND RSI < 50) |
| 실패 기록 처리 | 삭제 | 동일하게 기록·공개 |
| 시작 비용 | 대부분 유료 | 무료 (트랙레코드 쌓는 단계) |
| 거시 데이터 통합 | 없음 | VIX·FNG·Regime State 실시간 반영 |
| 멀티 전략 비교 | 없음 | 5개 전략 동시 경쟁 |

## 8. 리스크 및 대응

| 리스크 | 발생 가능성 | 대응 전략 |
|---|---|---|
| **알고리즘 전체 손실** (하락장 장기화) | 높음 | 투명 공개가 신뢰 자산, 손실 자체가 차별화 포인트 |
| **사용자 무관심** (트랙레코드 초기) | 매우 높음 | 1년치 데이터 쌓은 후 공개 → 부트스트래핑 회피 |
| **Binance 서울 접근 차단** | 중간 | EC2 Seoul 고정 IP, Upbit fallback 준비 |
| **금융업 규제** (투자 자문업) | 중간 | "알고리즘 연구 공개"로 포지셔닝, 투자 권유 표현 회피 |
| **경쟁자 복제** | 낮음 | 불변 타임스탬프 트랙레코드가 시간 자산, 복제 불가 |

### 규제 리스크 상세

- 한국 자본시장법상 투자자문업 등록 없이 구체적 투자 권유 금지
- 대응: "알고리즘 포워드 테스트 결과 공개"로 프레이밍
  - "이 알고리즘은 이 신호를 냈습니다" (사실 공개)
  - "이 신호대로 거래하세요" (투자 권유) — **금지**
- 18개월 내 실제 서비스 규모 성장 시 법률 자문 필수
