## 1. 시장 기회

### 크립토 시그널 시장 규모

- 전 세계 크립토 투자자 수: 약 4억 명 (2026 추산)
- 유료 시그널 서비스 이용자: 추정 1~3%
- 한국 크립토 투자자: 약 600~800만 명
- 관심 대상 (데이터 기반 시그널): 약 5~10만 명

### 포지셔닝

```
           데이터 투명성 낮음          데이터 투명성 높음

단순 예측    텔레그램 시그널 채널
             (대다수의 서비스)

거시+기술                              [ BTC Signal Arena ]
복합 분석                              (차별화 포지션)
```

기존 서비스는 왼쪽 아래에 몰려있다. 오른쪽 위 자리가 비어있다.

## 2. 수익화 타임라인

### 2026년 하반기: 무수익 (투자 단계)

- 비용: EC2 ~$15/월 + Supabase 무료 = **월 약 ₩20,000**
- 목표: 트랙레코드 쌓기, 아무도 보지 않아도 데이터는 쌓인다

### 2027년 상반기: 첫 수익화 시도

- 뉴스레터 + 웹 리더보드 오픈 (무료)

- 목표: **월 ₩100,000~500,000** (운영비 충당)

### 2027년 하반기: 유료 구독 정착

- 목표 구독자 100명, 월 ₩9,900 평균 ARPU
- 목표 수익: **월 ₩990,000**

### 2028년: 플랫폼 확장

- 구독자 500명 이상, 모델 다각화
- 목표 수익: **월 ₩5,000,000 이상**

## 3. 가격 전략

### 프리미엄 가격 설정 근거

```
경쟁 서비스 벤치마크:
  - 일반 텔레그램 시그널: ₩30,000~100,000/월
  - TradingView Pro: $14.95/월
  - 국내 퀀트 리포트: ₩50,000~200,000/월

BTC Signal Arena 포지셔닝:
  - 투명성 + 데이터 기반 = 프리미엄 정당화
  - 그러나 초기엔 낮게 진입 → 신뢰 후 가격 인상
```

### 단계별 가격 변화

| 단계 | 가격 | 이유 |
|---|---|---|
| Phase 1 (2026) | 무료 | 트랙레코드 없는 단계에서 유료화 불가 |
| Phase 2 초기 (2027 Q1) | 무료 + 조기 어답터 ₩4,900/월 | 얼리버드 전략 |
| Phase 2 안정 (2027 Q2) | ₩9,900 / ₩29,900 | 기본/프로 구분 |
| Phase 3 (2028) | ₩9,900 / ₩29,900 / ₩49,900 | API 티어 추가 |

## 4. 고객 획득 전략 (CAC = 0 목표)

### 콘텐츠 기반 자연 유입 (0원 전략)

1. **커뮤니티 기고**
   - 디시인사이드 코인갤, 클리앙, 네이버 카페
   - "6개월간 알고리즘 5개를 돌려봤더니..." 형태의 결과 공유
   - 광고 아님, 실제 데이터 기반 글

2. **트위터/X 트레이딩 커뮤니티**
   - 주간 알고리즘 성과 자동 트윗
   - 나쁜 결과도 솔직하게 공유 → 신뢰 구축

3. **블로그/브런치**
   - "이 알고리즘이 하락장에서 실패한 이유" 시리즈
   - 기술적 내용 + 데이터 공개 = 검색 유입

4. **Github 공개 (선택)**
   - 알고리즘 로직 공개 (코드)
   - "직접 돌려보고 싶은 개발자" 유입

### 바이럴 메커니즘

```
나쁜 결과 공개 → "와, 저 서비스는 손실도 공개하네" → 신뢰 → 공유
```

크립토 커뮤니티에서 투명한 손실 공개는 오히려 강력한 신뢰 신호가 된다.

## 7. 경쟁 분석

### 직접 경쟁자 (없거나 미약)

한국 시장에서 "거시 + 기술 지표 복합 + 투명 트랙레코드" 조합은 현재 공백 상태.

### 간접 경쟁자

| 서비스 유형 | 강점 | 약점 | 우리의 대응 |
|---|---|---|---|
| 텔레그램 시그널 채널 | 즉각적, 접근성 | 투명성 0, 트랙레코드 없음 | 투명성으로 차별화 |
| TradingView 아이디어 | 커뮤니티 규모 | 비체계적, 검증 없음 | 알고리즘 체계성 |
| 퀀트 서비스 (해외) | 기술 수준 높음 | 한국어 없음, 복잡 | 접근성 + 한국어 |
| 거래소 자체 신호 | 신뢰도 높음 | 편향 (거래 유도) | 중립성 |

### 방어 해자 (Moat)

```
시간이 지날수록 강해지는 단 하나의 해자:
"우리는 2026년 7월부터 단 하루도 빠지지 않고 기록했다"
```

이 트랙레코드는 복제 불가. 경쟁자가 오늘 시작해도 내년에야 6개월 데이터가 생긴다.

## 8. 출구 전략 (Exit Strategy)

당장 고려할 것은 아니지만 가능한 시나리오:

1. **자영업 수준 수익화** (가장 현실적)
   - 구독자 500~1,000명 유지, 월 ₩5,000,000~15,000,000
   - 개인 운영 또는 소규모 팀

2. **거래소/핀테크 인수**
   - 트랙레코드 + 사용자 기반 보유 시 M&A 타겟
   - Upbit, Bithumb 등 거래소의 프리미엄 기능 통합

3. **VC 투자 → 플랫폼화**
   - 알고리즘 마켓플레이스로 성장 후 Series A
   - 가장 불확실하지만 최대 upside
