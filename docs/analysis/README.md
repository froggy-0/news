# Analysis Docs

`docs/analysis/`는 특정 실행 결과, 장애/품질 분석, 사후 리뷰를 보관합니다.

## 하위 영역

| 경로 | 내용 |
| --- | --- |
| `sentiment-join/` | Sentiment Join 실행 리뷰, 진단, 개선 계획, baseline 문서 |
| `signal-improvement-audit-20260514.md` | 신호 개선 감사 |

## 사용 기준

- 날짜가 있는 실행 결과와 사후 리뷰는 이곳에 둡니다.
- 운영자가 반복적으로 따라야 하는 절차는 `docs/briefing/`, `docs/arena/operations/`, `docs/infrastructure/`로 승격합니다.
- 현재 코드 기준 요약은 `docs/reference/codebase-map.md`를 우선합니다.
