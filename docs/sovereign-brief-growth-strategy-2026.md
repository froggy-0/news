## 1. 프로젝트 현황 요약

### 강점 (실제로 작동하는 것)

| 항목 | 평가 |
|---|---|
| 완전 자동화 파이프라인 | 08:00 KST 일간 발행, 7단계 오케스트레이션 완성. 이 수준의 자동화를 개인이 구현한 것은 실질적 진입장벽 |
| 데이터 커버리지 | VIX + 금리 커브(10Y/2Y) + SOXX(반도체) + BTC ETF flow까지. 한국어 서비스 중 이 조합을 커버하는 곳 없음 |
| 개인화 인프라 존재 | Portfolio Impact Scoring 로직 이미 구현. 대부분의 경쟁자는 이 단계조차 없음 |
| Next.js SSG + Cloudflare | 정적 배포, 글로벌 CDN. 기술 스택 자체는 SEO에 유리한 구조 |

### 약점 (냉정한 평가)

**① 발견 가능성이 사실상 제로다.**
현재 이메일 단일 채널 운영은 "이미 아는 사람만 쓰는 서비스" 구조다. 이메일 뉴스레터는 성장 채널이 아니라 리텐션 채널이다. 신규 유입 경로가 없다.

**② 브랜드명 "Sovereign Brief"는 한국 시장에서 검색 불가다.**
한국 투자자가 이 이름을 검색할 이유가 없다. 네이버에서 "sovereign brief"를 치는 사람은 0명에 가깝다. 영문 브랜드명은 검색 유입을 완전히 포기하는 것과 같다.

**③ 개인화 "최대 2종목"은 차별점이 아니다.**
2종목 개인화는 기술적으로는 의미 있지만, 사용자가 가치를 인지하기 어렵다. "당신의 포트폴리오에 오늘 VIX가 미치는 영향"이라는 메시지가 없으면 그냥 뉴스레터다.

**④ 아카이브 페이지가 SEO 자산으로 활용되지 않는다.**
매일 발행되는 브리핑은 날짜별 URL로 존재하지만, 구조화 데이터·메타태그·내부 링킹이 없으면 구글은 이 페이지들을 인덱싱하지 않는다. 지금까지 발행된 수백 개의 브리핑이 SEO 관점에서 낭비되고 있다.

**⑤ Freemium 준비 중이지만 게이트 설계가 약하다.**
무엇을 막을 것인가가 명확하지 않으면 Freemium은 그냥 Free다.

## 2. Brainstorming 결과 — 전략 아이디어 Top 5

### #1. 텔레그램 채널 즉시 개설 (이메일보다 우선순위가 높다)

이메일 오픈율 20~25%, 텔레그램 오픈율 40~60%. 한국 투자자 커뮤니티는 텔레그램 채널 중심으로 운영된다. 텔레그램 채널은 바이럴 전파가 이메일보다 구조적으로 쉽다 — 채널 공유 한 번으로 수백 명에게 도달한다.

**실행:** 매일 08:00 브리핑의 요약 3줄 + 링크를 텔레그램에 자동 발행. 전체 내용은 이메일/웹사이트로 유도. 파이프라인에 텔레그램 bot 발행 1단계 추가하면 1~2일 작업이다.

**경고:** 텔레그램에 전체 콘텐츠를 무료로 뿌리면 이메일 구독 이유가 사라진다. 요약만 공개, 상세는 구독자 전용으로 게이트를 유지하라.

---

### #2. Freemium 게이트 재설계 — 지금 설계는 의미 없다

현재 방향(2종목 개인화)으로는 유료 전환이 일어나지 않는다. 사람들은 "더 많은 정보"보다 "더 빠른 정보"와 "내 자산에 직접적인 영향"에 돈을 낸다.

**권장 게이트 구조:**

| 티어 | 제공 내용 | 가격 |
|---|---|---|
| **Free** | 시장 요약 3줄 (VIX, 지수, BTC) — 텔레그램/웹 공개 | 무료 |
| **Standard** | 6개 섹션 전체 브리핑 + 아카이브 접근 | 월 9,900원 |
| **Pro** | 위 + 포트폴리오 임팩트 스코어 (최대 10종목) + 주간 리뷰 | 월 29,900원 |
| **Team** | Pro + Slack 연동 + 팀 5인 | 월 99,000원 |

Team 티어가 B2C 전체보다 ROI가 높다. VC, 스타트업 재무팀, 자산운용사 소규모 팀이 타겟이다.

---

### #3. 네이버 블로그 SEO — 구글보다 우선이다

한국 투자자의 정보 검색 경로: 네이버 > 유튜브 > 구글 순이다. 네이버 검색에서 공식 사이트 하나보다 블로그 포스트 10개가 노출 면적이 크다.

매일 생성되는 브리핑을 네이버 블로그에 요약본(500자)으로 자동 크로스포스팅하면:
- 네이버 검색 노출 누적
- 각 포스트가 "오늘의 미국 시장 요약 YYYY-MM-DD" 형태의 long-tail 키워드를 자연 포함
- 독자가 전체 내용을 보려면 sovereignbriefing.com으로 유도

파이프라인에 네이버 블로그 API 발행 단계 추가. 이것이 6개월 후 가장 큰 무기가 된다.

---

### #4. "오전 8시 시장 요약" 카테고리 선점 — 지금은 아직 비어있다

한국어로 미국 시장을 매일 아침 자동으로 요약하는 서비스는 사실상 없다. 가장 근접한 경쟁은 유튜브 채널(비자동화), 네이버 블로그(비체계), 인베스팅닷컴 한국어(번역 수준)다.

이 카테고리를 SEO 관점에서 선점하는 방법: "미국 증시 브리핑", "미국 시장 오전 브리핑"으로 모든 아카이브 페이지를 최적화한다. 1년치 날짜별 페이지가 있으면 이 키워드로 네이버·구글 모두에서 장기 트래픽을 가져온다.

**단, 6개월은 걸린다.** SEO는 즉각적인 결과가 없다. 이 전략은 지금 심어두는 씨앗이다.

---

### #5. B2B 슬랙 연동 — 개인 B2C보다 현실적인 수익 모델

```
개인 투자자 1,000명 × 9,900원  = 월 990만 원
기업 팀      33개   × 99,000원 = 월 990만 원 (같은 매출, 33분의 1의 고객)
```

스타트업 투자팀, VC 심사역, 자산운용사 소규모 팀은 이미 Slack을 쓴다. 매일 08:00 Slack 채널에 브리핑이 자동으로 올라오는 것은 B2B에서 명확한 가치다. 개인화(어느 회사 포트폴리오가 영향받는지)는 이 컨텍스트에서 훨씬 설득력이 있다.

Slack App Directory 등록까지 가면 이것 자체가 획득 채널이 된다.

## 3. SEO 전략

### 3-1. 핵심 키워드 (2026년 기준)

#### 1차 타겟 키워드 — 거래형/정보형

| 키워드 | 검색 의도 | 예상 월 검색량 | 경쟁도 |
|---|---|---|---|
| 미국 증시 오늘 | 정보형 | 높음 (5만+) | 매우 높음 |
| 미국 시장 요약 | 정보형 | 중간 | 중간 |
| VIX 지수 오늘 | 정보형 | 중간 | 낮음 |
| 미국 주식 뉴스레터 | 상업형 | 낮음 | 낮음 |
| 미국 증시 브리핑 | 정보형 | 낮음 | 낮음 ← **진짜 타겟** |

"미국 증시 오늘"은 이미 늦었다. 네이버, 인베스팅닷컴, 연합뉴스가 점령했다. 여기서 경쟁하지 마라.

#### 실제로 이길 수 있는 Long-tail 키워드

| 키워드 | 공략 방법 |
|---|---|
| `미국 증시 브리핑 YYYY-MM-DD` | 날짜별 아카이브 페이지가 자동으로 커버 |
| `VIX 지수 한국어 해석` | 설명형 가이드 페이지, 경쟁 없음 |
| `미국 10년물 금리 투자 영향` | 가이드 페이지 1개로 장기 트래픽 |
| `BTC ETF flow 의미` | 극소수 한국어 콘텐츠 존재 |
| `공포탐욕지수 투자 전략` | 정보 페이지 작성 시 상위 가능 |
| `반도체 ETF SOXX 분석` | SOXX 커버리지는 국내 서비스 중 거의 없음 |
| `미국 주식 매일 요약 구독` | 상업형, 전환율 높음 |

### 3-2. Landing Page 구조 (홈페이지 개편)

**권장 홈페이지 구조:**

```
[H1] 매일 오전 8시, AI가 정리한 미국 시장 브리핑
      → "미국 증시 브리핑", "미국 시장 요약" 키워드 자연 포함

[Section 1] 오늘의 핵심 지표 (실시간 표시)
      VIX / US10Y / SPY / BTC / Fear&Greed
      → 각 항목에 설명 텍스트 → 키워드 밀도 확보

[Section 2] 최근 브리핑 샘플 (최신 1개 전문 공개)
      → 신규 방문자의 가치 즉시 확인
      → 매일 업데이트 → 크롤러 재방문 유도

[Section 3] 무엇이 다른가
      → "Portfolio Impact Score": 내 보유 종목이 오늘 뉴스에 얼마나 영향받는가
      → "6개 섹션 체계": 거시경제 → 빅테크 → 반도체 → 비트코인 → 시장신호 → 전망

[Section 4] 구독 CTA
      → Free tier 먼저 (이메일 + 텔레그램 선택)
```

**홈페이지 메타 태그:**
```html
<title>Sovereign Brief — 매일 오전 8시 미국 증시 브리핑 | AI 시장 분석</title>
<meta name="description" content="VIX, 금리, SPY, QQQ, 비트코인을 매일 아침 한국어로 정리합니다.
AI가 생성한 미국 시장 브리핑을 이메일로 받아보세요. 무료 구독.">
```

**아카이브 페이지 메타 태그 패턴 (날짜별 자동 생성):**
```html
<title>2026년 4월 1일 미국 증시 브리핑 — VIX, 금리, 비트코인 요약</title>
<meta name="description" content="2026년 4월 1일 미국 시장 요약.
VIX [값], 10년물 금리 [값], SPY [값]. AI 분석 포함.">
```

이 패턴을 아카이브 전체에 자동 적용하면 날짜별 페이지가 long-tail 키워드 기계가 된다.

### 3-3. Schema Markup

**아카이브 페이지 (NewsArticle):**
```json
{
  "@context": "https://schema.org",
  "@type": "NewsArticle",
  "headline": "2026년 4월 1일 미국 증시 브리핑",
  "datePublished": "2026-04-01T08:00:00+09:00",
  "dateModified": "2026-04-01T08:00:00+09:00",
  "publisher": {
    "@type": "Organization",
    "name": "Sovereign Brief",
    "url": "https://sovereignbriefing.com"
  },
  "description": "오늘의 VIX, 금리, SPY, BTC 요약 및 포트폴리오 임팩트 분석",
  "inLanguage": "ko-KR"
}
```

**가이드 페이지 (Article + FAQ):**
```json
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "VIX 지수란 무엇인가 — 투자자가 알아야 할 모든 것",
  "datePublished": "2026-04-01",
  "author": {
    "@type": "Organization",
    "name": "Sovereign Brief"
  }
}
```

아카이브 페이지가 NewsArticle 스키마를 갖추면 구글 뉴스 인덱싱 대상이 된다. 일반 SEO보다 노출 배수가 다르다.

### 3-4. 콘텐츠 로드맵 90일

#### Week 1~2: 기술 SEO 기반 세팅 (이것부터 하지 않으면 나머지 의미 없다)

- [ ] 아카이브 페이지 전체 메타태그 자동 생성 (`generateMetadata` — Next.js App Router)
- [ ] sitemap.xml 아카이브 페이지 포함 여부 확인 및 수정
- [ ] 브리핑 페이지에 NewsArticle 스키마 마크업 추가
- [ ] Google Search Console 등록 및 기존 URL 인덱싱 요청
- [ ] robots.txt 확인 (아카이브 페이지 크롤링 허용 여부)
- [ ] Open Graph 태그 최적화 (카카오톡 공유 미리보기용)

#### Week 3~4: 핵심 가이드 페이지 3개 작성

| URL | 타겟 키워드 | 콘텐츠 방향 |
|---|---|---|
| `/guide/vix-지수-투자-영향` | VIX 지수 한국어 해석 | VIX란 무엇인가, 수치별 해석, 투자 판단에 활용하는 법 |
| `/guide/미국-금리-커브-해석` | 미국 10년물 금리 투자 영향 | 10Y/2Y 스프레드, 역전 의미, 주식/채권 영향 |
| `/guide/btc-etf-flow-분석` | BTC ETF flow 의미 | ETF 자금 흐름 읽는 법, 가격과의 관계 |

각 가이드 페이지 하단에 브리핑 구독 CTA 포함. 이 3개가 pillar 콘텐츠 역할.

#### Week 5~8: 네이버 블로그 채널 개설 + 자동화

- 네이버 블로그 개설
- 파이프라인에 네이버 블로그 자동 포스팅 단계 추가
- 포스팅 형식: 제목 `"YYYY-MM-DD 미국 증시 브리핑 — 오늘 핵심 3가지"` + 500자 요약 + 원문 링크
- 주 5회 이상 발행 → 네이버 검색 신뢰도 축적

#### Week 9~12: 텔레그램 + 교차 트래픽 완성

- 텔레그램 채널 개설 및 자동 발행 파이프라인 연동
- 텔레그램 채널 설명에 웹사이트 + 이메일 구독 링크
- 네이버 블로그 포스트 하단에 텔레그램 채널 링크 포함
- 텔레그램 → 웹사이트 → 이메일 구독 퍼널 완성

## 6. ROI 판단

### 종합 점수: **62/100**

#### 잘하면 80점이 될 수 있는 이유

- 기술 인프라가 이미 완성됐다. 경쟁자가 이 수준의 자동화를 구축하려면 6개월~1년이 필요하다.
- 한국어 미국 시장 브리핑은 실제로 수요가 있고 공급이 없는 틈새다.
- 데이터 커버리지(SOXX, BTC ETF flow)는 국내 어디에서도 이 조합으로 한국어 분석을 안 한다.
- Next.js SSG 구조는 SEO에 최적화된 스택이다. 메타태그만 붙이면 즉시 인덱싱 가능.

#### 지금 당장 62점인 이유

- **발견 가능성이 없다.** 아무도 이 서비스의 존재를 모른다. 지금은 구독자에게만 가치 있는 서비스다.
- **이메일만으로는 성장하지 않는다.** 채널 확대를 준비 중이지만, 텔레그램 채널이 내일 당장 없다는 것이 문제다.
- **Freemium 게이트가 확정되지 않았다.** 무엇을 막을지 결정하지 않으면 Freemium은 운영 비용만 늘리는 Free 서비스가 된다.

#### 이건 의미 없다 (하지 말 것)

- **구글 SEO로 단기 트래픽 확보:** 6개월 이상 기다려야 한다. 지금 당장 성장이 필요하다면 구글 SEO는 지원 전략이지 주전략이 아니다.
- **B2C 개인 구독자 1만 명 목표:** 월 9,900원으로 1만 명이면 9,900만 원/월이다. 1년 내 현실적으로 불가능하다.
- **인스타그램/X(트위터) 채널:** 투자 정보는 텍스트가 핵심이고 한국 투자자는 텔레그램과 네이버를 쓴다. 인스타는 낭비다.
- **유튜브 채널:** 영상 제작 비용 대비 초기 ROI가 없다. 구독자 1,000명 이후에나 고려하라.

#### 지금 당장 해야 하는 것 3가지 (이것만 해도 점수가 75점이 된다)

```
1. 텔레그램 채널 개설 + 파이프라인 연동
   공수: 1~2일 / 효과: 즉각적

2. 아카이브 페이지 메타태그 자동화 (generateMetadata)
   공수: 반나절 / 효과: 지금까지 발행한 모든 브리핑이 SEO 자산으로 활성화

3. Freemium 게이트 확정 (무엇을 막을지 결정)
   공수: 코드 작업 아님, 의사결정 1시간
```

이 셋을 2주 안에 완료하면 나머지 전략의 실행 기반이 생긴다. 순서가 틀리면 SEO 콘텐츠를 만들어도 전환할 제품이 없거나, 제품이 있어도 아무도 발견하지 못하는 상태가 반복된다.
